Instalação:

1) Extraia o ficheiro para um diretório.

2) Abra a terminal no diretório onde o programa esta presente.

3) Escreva os códigos ‘gcc Servidor.c –o Servidor’ e ‘gcc Cliente.c –o Cliente’ para obter os executáveis.

4) Escreva ‘./Servidor’ para executar o programa. O utilizador do gestor é ‘gestor’ e a palavra-passe é ‘admin1’.

5) Crie um utilizador (quando lhe for pedido o nome escreva só um nome pois se escrever mais que uma palavra o utilizador não será criado como é entendido).

6) Escreva './Cliente' para executar o programa como cliente onde lhe é pedido o nome de utilizador e a palavra-passe que criou no programa do servidor.

Reinstalação:

1) Remova todos os ficheiros exceto 'Servidor.c', 'Cliente.c' e 'stats_backup.txt' .

2) Copie o ficheiro 'stats_backup.txt' e mude o nome da copia para 'stats.txt'.

3) Volte a compilar os programas.

Notas:

1) Por convenção este programa so permite a criação de 100 produtos no máximo.

2) O input durante os menus tem de ser um algarismo ou o programa deixa de funcionar.
